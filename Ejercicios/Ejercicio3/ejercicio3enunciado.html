Ejercicio 3

Introduccion

El objetivo de este ejercicio es practicar con las hojas de estilo y aprender nuevas reglas CSS

Tareas

Crear una hoja de estilo para modificar el aspecto de la página existente Tarta de Calabaza que muestre los siguientes cambios

  • Hacer que los titulos (encabezados) y el texto general tengan tipos de letras (familias) distintos
  • Hacer que el texto de la introducción aparezca en cursiva
  • Añadir un color de fondo a la página web
  • Mediante CSS hacer que la imagen sea más pequeña
  • Añadir bordes a las listas. Se puede investigar para que cada lista tenga un tipo de borde distinto

Cosas para subir nota:

Encontrar en Internet (en una referencia o en otras páginas de ejemplo) cosas que no hemos visto para hacer cosas creativas
