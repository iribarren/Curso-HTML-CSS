Ejercicio 1

Introduccion

El objetivo de este ejercicio es consolidar lo aprendido hasta ahora sobre los siguientes elementos:

  • Encabezados
  • Parrafos
  • Enlaces
  • Imagenes
  • Formularios

Tareas

Realizar un sitio web (un conjunto de paginas html) con las siguientes caracteristicas:

  1. Página de inicio. La página de inicio debe contener una lista con enlaces al resto de las páginas
  2. Página con galeria de fotos. Una página con 5 imagenes distintas
  3. Página de contacto. Una página desde la que una persona pueda enviar un comentario a traves de un formulario. Además debe dar su información personal
  4. Página en la que se pueda ver una receta. Intentar utilizar todos los elementos vistos hasta ahora

Cosas para subir nota:

  • Que desde todas lás páginas se pueda volver a la página principal o
  • Que todas las paginas esten enlazadas
  • Que las fotos de la galeria, esten en su propia carpeta separada
  • Inspeccionar este fichero para saber como dar enfasis al texto (se puede consultar internet también para ver como se pone la negrita)
